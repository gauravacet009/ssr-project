### Installing

First clone project and install dependencies

```sh
$ git clone https://github.com/gaurav/ssr-project.git
$ npm install
```

Run on local

```sh
$ npm run dev
```

Navigate to [http://localhost:3000](http://localhost:3000)

## Deployment

Deployment build

```sh
$ npm run build:prod
```
